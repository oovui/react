# 快速上手

## 安装 

Install `@oovui/core` and `styled-components` from npm:

`npm install @oovui/core styled-components`

## 基本用法


```js
import { Button } from '@oovui/core'

const App = () => (
  <Button primary>Hello world!</Button>
)

export default App
```
