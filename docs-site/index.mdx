# @oovui/core

[![npm](https://img.shields.io/npm/v/@oovui/audio-player-react.svg)](https://www.npmjs.com/package/@oovui/audio-player-react)  [![JavaScript Style Guide](https://img.shields.io/badge/code_style-standard-brightgreen.svg)](https://standardjs.com)


<ImgBox src="https://github.com/oovui/react-components/blob/master/docs-site/images/logo.png?raw=true"/>


@oovui/react 是一套基于 React.js 和 Styled Components 💅 . 的开源 UI 组件库。

## 特性
基于React.js 16.x 稳定版

所有组件基于ES6开发,使用  babel + rollup 构建

区分核心组件和业务组件,`@oovui/core` 只包含核心组件，做到最小的包

业务组件可以根据需要安装，按需导入

## 兼容性

IE9+ 及符合W3C web标准的现代浏览器

部分组件支持IE11及以上浏览器，具体见组件使用说明。

## 贡献

如果你在使用时遇到问题，或者有好的建议，欢迎给我们提 Issue 或 Pull Request
